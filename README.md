# 프로젝트 이름
tomaDO
## 프로젝트 소개
뽀모도로와 게이미피케이션을 이용한 집중력 동기부여 서비스
## 개발 기간
* 24.03.17~24.03.25
## 멤버
*  김수영:기획
*  이아영:디자인
*  최규원, 이하빈:프론트엔드
*  박소연, 신주은:백엔드
## 개발 환경
* Front - Flutter, JAVA
* Back - Spring, MySQL, AmazonEC2
